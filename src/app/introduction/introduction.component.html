WELCOME TO STOCKS DEMO APP

This application is written in Angular 4 and .NET Core 2.0

It's deployed on Azure as two separate Docker containers. One for front-end static web app and one for back-end API.

In the input field below, you can type a number between 1 and 20 which will determine the number of stocks generated for this demo session.
